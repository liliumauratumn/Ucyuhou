# Final Report Formats

## Implementation task

```md
**Done:** <feature/change> implemented and verified.

**What changed**
- `<file>` — <change>
- `<file>` — <change>

**Verified**
- `<command>` — passed
- `<manual check>` — observed <result>

**Decisions made**
- <decision and reason>

**Noticed but not changed**
- <out-of-scope observation>
```

## Partial completion / blocked

```md
**Partially done:** <summary>.

**Complete and verified**
- <item> — <check result>

**Blocked**
- <blocked item>
- Reason: <reason>
- What would unblock it: <needed info/permission>

**State of the code**
- <green/partial/reverted>
```

## Review task

```md
**Review:** <number> bug(s), <number> risk(s), <number> nit(s).

**Bug**
- `<file:line>` — <issue>. Why it matters. Suggested fix.

**Risk**
- `<file:line>` — <issue>. Why it matters. Suggested fix.

**Nit**
- `<file:line>` — <style issue>.

No code changes were made.
```

## Stopped for ambiguity or risk

```md
**Stopped before acting:** <reason>.

**Task**
- <task>

**What I found**
- <facts>

**Options**
1. <option>
2. <option>

**Recommendation**
- <recommended option and why>

**Nothing has been changed yet.**
```
