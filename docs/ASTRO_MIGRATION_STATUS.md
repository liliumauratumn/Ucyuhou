## 4. 次の予定

Phase 7の編集基盤は実装済みである。次は大量生成ではなく、1本の記事で次の全工程を試し、必要な修正だけを編集規則へ戻す。

```text
企画書 → 資料調査 → 原稿 → 画像 → 校閲 → Astro検証
```

この試行が通れば、同じ工程を今後の記事制作へ反復できる。Git commit、push、公開はユーザーの明示指示に従って別途行う。
